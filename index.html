Time Wheel

Tasks in Chronological Order

Instructions

  1. Set Times: Input the start and end times for your task. Tip: Slices can cross midnight (e.g., 22:00 to 05:00)!
  2. Task Details: Enter the task name. Click the color box directly next to the input to open the browser color picker and select a custom color.
  3. Place on Wheel: Click the Add button to draw the task sector on the Time Wheel.
  4. Auto-Sorting: Slices are automatically sorted in chronological order under the task list.
  5. Edit Tasks: Click the pencil icon (✎) in the list to load a task back into the form. Modify details and click Add to save changes.
  6. Delete Tasks: Click the cross icon (✕) to remove a task from the list and wheel.
  7. Save/Load: Use 📤 Export to download your schedule as a JSON file, or 📥 Import to restore a saved schedule.
  8. Print Layout: Click Print to save the Time Wheel visually (hides controls and instructions).
